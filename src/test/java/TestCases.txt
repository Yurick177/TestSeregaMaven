                              Тест-кейс №1 Позитивный

Зоголовок:       Регистрация на сайте

Предусловие:    заходим на сайт http://the-internet.herokuapp.com/login, логинимся.

Шаги (Ш) и Ожидаемый результат (ОР):

Ш1. Перейти на сайт http://the-internet.herokuapp.com/login ОР1 успешно открыто окно авторизации

Ш2. Ввести значение в поле “Username” состоящее из латинских букв "tomsmith" ОР2  В поле “Username” отображается введённое имя

Ш3. Ввести значение в поле “Password” состоящее из латинских букв "SuperSecretPassword!" ОР3 В поле “Password” пороль не отображается(точки)

Ш4. Нажимаем на форму  "Login" ОР4 успешно авторизовались

Ш5. Нажимаем на форму  "Logout" ОР5 выходим из авторизации
